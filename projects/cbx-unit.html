CBX Unit

Description

  • CBX Unit

  • 2018

Developed a Minimum Viable Product (MVP) for CBX Unit cryptocurrency using Laravel and Python. The project involved setting up a decentralized network for ledgering using multiple Raspberry Pi devices, creating a distributed system for blockchain operations.

The solution combined web application development with hardware integration, utilizing Raspberry Pi devices to establish a decentralized network infrastructure that supports secure and distributed ledger management for the cryptocurrency platform.

  • Cryptocurrency MVP development
  • Decentralized network architecture
  • Raspberry Pi hardware integration
  • Distributed ledger system
  • Laravel web application
  • Python backend services
  • Blockchain infrastructure
