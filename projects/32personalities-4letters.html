4-Letters - Personality Advice App

Description

  • 32 Personalities

  • 32Personalities.com

  • 2022 - 2023

Built the backend and API infrastructure for 4-Letters, a Myers-Briggs personality assessment and advice application. The system provides personalized recommendations and insights based on user personality types, requiring robust data processing and API endpoints.

The project included comprehensive server management services, ensuring reliable performance and secure handling of user data and personality assessment results.

  • Personality assessment backend
  • RESTful API development
  • Data processing algorithms
  • User recommendation system
  • Server management and maintenance
  • Data security implementation

  • MySQL
  • Linux
