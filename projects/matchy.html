I developed the Matchy Uniform website and online store using Laravel, incorporating a custom ordering system for tailored bulk orders. Additionally, I built the API to support app integration, ensuring a seamless experience across platforms.

The project also included comprehensive infrastructure setup, including .SA domain registration, Cloudflare configuration for enhanced security and performance, official email setup, and Google Business Profile setup to establish a strong online presence.
