Najah Association - Internal Portal

Description

  • Najah Association

  • 2024 - 2026

Developed a comprehensive internal portal for Najah Association using Laravel. The web application provides various features for employee management, task tracking, and departmental operations.

Features

  • User authentication and authorization
  • User profile management
  • Dashboard for quick overview
  • Department management
  • Executive plan tracking
  • Task management with approval system
  • Internal messaging system
  • Request management
  • Role-based access control
  • Form builder and management
  • Calendar view for tasks
  • File attachments for tasks and messages

Technology

  • Laravel
  • PHP
  • MySQL
  • JavaScript
  • HTML
  • CSS
