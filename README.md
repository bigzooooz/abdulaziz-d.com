## About This Repo

It's a single-page portfolio. All my content lives in `data.json` because I like to keep things simple. When I need to update something, I just edit that one file. The site reads it and does the rest. Easy peasy.

I used the [BreezyCV template](https://themeforest.net/item/breezycv-resume-cv-template/26081195) as a starting point for the design, but then I heavily modified how it works under the hood. The looks stayed pretty similar, but the whole content loading system is my own - JSON-driven and much easier to maintain. No frameworks, no overthinking. Just works.
